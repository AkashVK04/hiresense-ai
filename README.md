# 🚀 HireSense AI

An AI-powered Mock Interview & Resume Analysis Platform that helps students prepare for technical interviews using Generative AI.

## 🌐 Live Demo

https://hiresense-ai-iicu.vercel.app/

## ✨ Features

- 🔐 Google Authentication
- 📄 Resume Upload (PDF)
- 🤖 AI Resume Analysis
- 💼 Role-based Interview Questions
- 🧠 AI Answer Evaluation
- 📊 Performance Analytics
- 📈 Dashboard
- 📑 PDF Report Generation
- ☁️ Firebase Firestore Storage
- 🚀 Vercel Deployment

## 🛠️ Tech Stack

Frontend
- React.js
- Vite
- Tailwind CSS
- Framer Motion

Backend
- Vercel Serverless Functions

AI
- Groq API (Llama)

Database
- Firebase Firestore

Authentication
- Firebase Authentication

Charts
- Recharts

Deployment
- Vercel
